# Hero-species reference boards

Create one local folder for each hero species: `rose`, `poppy`, `lily`,
`sunflower`, `orchid`, and `lotus`. Each board should cover front, side,
underside, petal detail, center detail, leaf, stem, and growth stages, ideally in
both studio and outdoor light.

For every photograph, record the following in a local `sources.csv`:

```text
species,view,filename,title,creator,source_url,license,license_url,retrieved_at,notes
```

Prefer public-domain or CC BY material from museums, herbaria, universities, or
the original photographer. Do not use images with unclear provenance, and do
not assume that search-engine thumbnails are licensed for reuse. The reference
folders and `sources.csv` files are ignored by Git until their redistribution
rights have been reviewed.
